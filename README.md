# Research Paper Summarizer

This project is a modular LLM-based application designed to summarize research papers. It uses a structured system prompt and a multi-module architecture to ensure accuracy, safety, and audience-specific tailoring.

## Modules
* **01 Intake:** Handles text normalization.
* **02 Section Loop:** Processes individual sections.
* **03 Guardrails:** Prevents hallucinations and checks text quality.
* **04 Rendering:** Formats the final output.
* **05 Citation Extractor:** Identifies references.
* **06 Key Contributions:** Highlights novel findings.
